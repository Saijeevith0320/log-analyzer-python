# Automated Log Analyzer & Suspicious Activity Detector (Python SOC Tool)

A Python-based security automation tool that scans Linux system logs to detect suspicious activity.
This project demonstrates practical SOC (Security Operations Center) automation skills useful for cybersecurity roles.

## Features
- Detects failed SSH login attempts  
- Flags invalid user access attempts  
- Detects high-risk root logins  
- Extracts suspicious IP addresses  
- Generates CSV-based security report  

## Usage
Run the script:
```
python log_analyzer.py
```

Enter the path to your log file:
```
sample_logs.txt
```

A `security_report.csv` will be generated.
